# web-mariscos-prototype

Landing page de restaurante de mariscos migrada a Astro + React, con animaciones, menú filtrable y formulario de reservaciones.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Variables opcionales

Si quieres activar el envío de correos para reservaciones con EmailJS, define estas variables públicas:

- `PUBLIC_EMAILJS_PUBLIC_KEY`
- `PUBLIC_EMAILJS_SERVICE_ID`
- `PUBLIC_EMAILJS_TEMPLATE_CLIENT`
- `PUBLIC_EMAILJS_TEMPLATE_BUSINESS`
- `PUBLIC_BUSINESS_EMAIL`
